中新网白山10月24日电(高龙安 吕盛楠)10月24日，以“白山松水·一路长红”为主题的吉林省第二届红色旅游节活动在白山临江举办，以此持续推动该省红色旅游高质量发展，宣传红色旅游资源，拓展红色旅游市场。 吉林省文化和旅游厅副厅长宋薇表示，该省高度重视红色资源保护利用工作，提炼形成“三地三摇篮”(东北抗日联军创建地、东北解放战争发起地、抗美援朝后援地、新中国汽车工业的摇篮、新中国电影事业的摇篮、中国人民航空事业的摇篮)的红色标识，并将“红色资源优势”作为吉林省“六大发展优势”之一摆在重要位置。 2021年以来，吉林省文化和旅游厅抽调宣讲员走进不同场所，举办百余场红色故事宣讲活动。此次红色旅游节，回顾了几年来红色故事宣讲活动的精彩片段，现场宣布了吉林省红色故事宣讲团正式成立，并聘请杨靖宇将军曾孙马铖明为吉林省红色文化旅游推广大使。 活动中，发布了9条吉林省红色旅游融合线路产品。这些线路是在充分调研红色旅游市场需求基础上，通过“红色+”的方式，将红色景区与生态、工业、乡村等旅游资源巧妙融合，为游客呈现一个多彩的吉林。
