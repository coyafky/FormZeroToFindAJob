Float Left

This text will wrap around the floated element to the left. By applying `float-left`, the text flows around the box as expected.

Float Right

This text will wrap around the floated element to the right. By applying `float-right`, the text flows around the box in a way that keeps the floated element on the right.

Float None

This element has `float-none`, so it does not float and the text does not wrap around it.
